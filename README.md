# 釣り大会アプリ 画面設計（PlantUML）

このリポジトリは、釣り大会アプリの **画面遷移図・画面設計** を PlantUML で管理するためのものです。  

## 画面遷移図の方針
- 入口は **ホーム** から開始
- **大会画面（Tournament）** はコンテナとして扱い、内部に
  - 概要
  - 参加者
  - 検寸（魚登録/一覧）
  - ランキング
  - 設定
  などのサブ画面を持つ想定
- 編集/削除などの操作は「詳細 → 編集」や「削除確認」を挟む

## 更新ルール（運用）
- 変更は原則 `.puml` を更新
- 必要に応じてエクスポート画像も更新（任意）
